# 🥗 Smart Nutrition Meal Planner

## 📌 Project Description
The **Smart Nutrition Meal Planner** is a data-driven system that generates personalized meal plans based on a user's health profile, dietary restrictions, and lifestyle habits. The system analyzes attributes such as age, BMI, physical activity level, medical conditions, and dietary preferences to recommend balanced and nutritious meals.

This project demonstrates the application of **Data Science, Machine Learning, and Data Visualization** to improve health and nutrition planning.

---

## 🎯 Objectives
- To provide personalized meal recommendations.
- To analyze health and nutrition data using machine learning techniques.
- To assist users in maintaining a balanced diet.
- To build a data-driven nutrition recommendation system.

## ⚙️ Project Workflow

### 1️⃣ Data Collection
The dataset containing user health information and dietary preferences is collected.

### 2️⃣ Data Preprocessing
- Handling missing values
- Removing duplicates
- Data normalization
- Feature selection

### 3️⃣ Exploratory Data Analysis (EDA)
Visualization techniques are used to understand patterns in:
- BMI distribution
- Caloric intake
- Disease types
- Activity levels

### 4️⃣ Model Training
Machine learning models are used to analyze user health data and generate recommendations.

Algorithms used:
- Decision Tree
- K-Nearest Neighbors (KNN)
- Random Forest

### 5️⃣ Meal Recommendation
Based on the trained model, the system predicts and recommends suitable meal plans for users.
